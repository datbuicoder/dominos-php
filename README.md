# Online Fast Food Ordering System

A web-based fast food ordering system built with PHP and MySQL.  
This project allows users to browse food items, place orders, and manage accounts.

## Tech Stack
- Backend: PHP (mysqli)
- Frontend: HTML, CSS, JavaScript, Bootstrap 4
- Database: MySQL
- Tools: XAMPP, phpMyAdmin, Composer

## Features
- User authentication (login / register)
- Food menu browsing
- Order management
- Admin dashboard for managing products and orders
